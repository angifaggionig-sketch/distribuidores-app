## 📋 Descripción

**Snackora ERP** es una aplicación web desarrollada para digitalizar y optimizar los procesos internos de microempresarios dedicados a la distribución y ventas. El sistema permite gestionar ventas, controlar inventario y generar reportes de gestión desde cualquier dispositivo con acceso a internet.

Este proyecto nació como una solución real a la necesidad de pequeños negocios de contar con herramientas tecnológicas accesibles para mejorar su eficiencia operativa.

---

## ✨ Funcionalidades

- 🔐 **Login de usuarios** — Autenticación segura con control de acceso
- 🛍️ **Registro de ventas** — Ingreso y seguimiento de transacciones comerciales
- 📦 **Control de inventario** — Gestión de productos y stock disponible
- 📊 **Reportes y estadísticas** — Visualización de datos con gráficos interactivos

---

## 🚀 Tecnologías utilizadas

| Tecnología | Uso |
|---|---|
| **React 19** | Interfaz de usuario |
| **React Router** | Navegación entre vistas |
| **Supabase** | Base de datos y autenticación |
| **Recharts** | Gráficos y visualización de datos |
| **Vite** | Entorno de desarrollo |
| **JavaScript** | Lenguaje principal |

## ⚙️ Instalación local

```bash
# Clona el repositorio
git clone https://github.com/angifaggionig-sketch/distribuidores-app.git

# Entra a la carpeta
cd distribuidores-app

# Instala las dependencias
npm install

# Inicia el servidor de desarrollo
npm run dev
```
